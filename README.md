# 🛤️ 공유 앨범 추억 로드맵 앱 WithLive

<br>

## 📋 프로젝트 소개

- ✈️ 친구, 연인 또는 가족끼리 추억을 회상하는 추억 공유 앨범 앱입니다.
- 💑 특정 사람들과 어떤 이벤트가 있었는지 시간 순으로 볼 수 있습니다.
- 📷 이벤트마다 공유 앨범으로 친구와 사진을 쉽게 공유할 수 있습니다.
